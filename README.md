## 🚀 Quick Start

### Prerequisites

- Node.js 16.8+
- npm or yarn

### Installation Magic ✨

```bash
# Clone the repository
git clone https://github.com/saadelnady/ecommerce-task.git

# Navigate to the project
cd dubisign-ecommerce

# Install dependencies
npm install
# or if you prefer yarn
yarn install

# Start the development server
npm run dev
# or
yarn dev
```
